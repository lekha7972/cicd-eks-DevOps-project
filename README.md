🚀 CI/CD Pipeline using GitHub Actions, EKS, Terraform & Helm

This project demonstrates a production-style CI/CD pipeline that automates building, containerizing, and deploying an application to AWS EKS.

---

🔧 Tech Stack

- AWS (EKS, ECR, IAM, VPC)
- Terraform (Infrastructure as Code)
- Docker (Containerization)
- Kubernetes (Orchestration)
- Helm (Kubernetes package manager)
- GitHub Actions (CI/CD)

---

🎯 Project Goal

To design and implement a complete DevOps pipeline:

- Automate build and deployment
- Use Infrastructure as Code
- Deploy containerized apps on Kubernetes
- Follow real-world DevOps practices
